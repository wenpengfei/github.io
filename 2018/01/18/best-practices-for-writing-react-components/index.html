我们编写 React 组件的最佳实践

2018年1月18日

刚接触 React 的时候，在一个又一个的教程上面看到很多种编写组件的方法，尽管那时候 React 框架已经相当成熟，但是并没有一个固定的规则去规范我们去写代码。

在过去的一年里，我们在不断的完善我们的做法，直到满意为止。

本文会列出我们自己在使用的最佳实践，不管你是刚入门的新手还是很有经验的开发者，我们都希望本文对你有所帮助。

开始之前，先列几条：

  • 我们使用ES6/ES7
  • 如果你无法区分页面组件和容器组件，推荐阅读 这篇文章（需要梯子）
